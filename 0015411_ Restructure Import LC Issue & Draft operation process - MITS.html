Instruction to fix Transaction Unhandled Error due to empty account, currency

in TRANSACTION_TEMPLATE table on ACCOUNT_CURRENCY_EXP column handle null value for all account expression of respective TR_CODE.

example 
previous (shows unhandled error):
data.getPaidBillSplittingInfo().getInvestmentAccountCurrency() 
data.getPaidBillSplittingInfo().getFcHeldAccountCurrency()
data.getPaidBillSplittingInfo().getErqAccountCurrency()
data.getPaidBillSplittingInfo().getChargeAccountCurrency()
data.getPaidBillSplittingInfo().getCustomerAccountCurrency()

solution : 
data.getPaidBillSplittingInfo().getInvestmentAccountCurrency() == null ? "BDT" : data.getPaidBillSplittingInfo().getInvestmentAccountCurrency()
data.getPaidBillSplittingInfo().getFcHeldAccountCurrency() == null ? "BDT" : data.getPaidBillSplittingInfo().getFcHeldAccountCurrency()
data.getPaidBillSplittingInfo().getErqAccountCurrency() == null ? "BDT" : data.getPaidBillSplittingInfo().getErqAccountCurrency()
data.getPaidBillSplittingInfo().getChargeAccountCurrency() == null ? "BDT" : data.getPaidBillSplittingInfo().getChargeAccountCurrency()
data.getPaidBillSplittingInfo().getCustomerAccountCurrency() == null ? "BDT" : data.getPaidBillSplittingInfo().getCustomerAccountCurrency()
